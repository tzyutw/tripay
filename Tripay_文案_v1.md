## S-04｜新增／編輯消費（Sheet）

**Sheet 標題**

| 模式 | 標題 |
|------|------|
| 新增 | 記一筆 |
| 編輯 | 編輯消費 |

**欄位標籤與 Placeholder**

| 欄位 | 標籤 | Placeholder |
|------|------|-------------|
| 標題 | 什麼花費？ | 例如：午餐 🍜 |
| 外幣金額 | 外幣金額 | 稍後再填也可以 |
| 台幣金額 | 台幣金額 | 稍後再填也可以 |
| 付款方式 | 怎麼付的？ | 選擇付款方式 |
| 日期 | 日期 | 選擇日期 |
| 支付者 | 誰請客？ | 選擇付款人 |
| 費用類型 | 費用類型 | — |
| 分帳成員 | 分給誰？ | — |
| 各自金額 | 各自多少？ | 輸入金額 |

**付款方式選項**

| 選項 | 文案 |
|------|------|
| cash | 現金 |
| credit_card | 信用卡 |
| stored_value | 儲值卡 |

**費用類型選項**

| 選項 | 文案 | 說明 |
|------|------|------|
| 共同 | 大家一起分 | 勾選成員，等比分攤 |
| 各自 | 各自付各自的 | 展開每人個別金額 |
| 自付 | 只算我自己 | 不進入分帳計算 |

**即時提示**

| 情境 | 文案 |
|------|------|
| 共同模式，每人金額 | 每人 $X |
| 各自模式，加總比對（正常） | 已填 $X／總額 $X ✓ |
| 各自模式，加總比對（不符） | 已填 $X／總額 $X，差 $Y |

**操作按鈕**

| 按鈕 | 文案 |
|------|------|
| 儲存 | 記下來 |
| 取消 | 取消 |
| 刪除（編輯模式） | 刪除這筆 |

**刪除確認 Dialog**
> 刪除後無法復原。
> 主按鈕：刪除　／　次按鈕：取消
