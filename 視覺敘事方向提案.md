## ✅ 定案方向：旅遊日記（精修版）

**核心定義**：暖色調 + 精緻無手寫感，日記的溫度用配色傳達，不靠字型

| 設計決策 | 規格 |
|---------|------|
| 字型 | Inter（全介面統一，清晰精緻） |
| Header 背景 | #7C2D12（深暖棕，皮革日記質感） |
| 頁面底色 | #FFFBEB（暖米白） |
| 主文字 | #1C0A00（深暖黑） |
| 輔助文字 | #92603A（中棕） |
| 外幣標示 | #B8977A（淺棕，層次低於台幣） |
| 「待填」狀態 | #C2410C 橘紅左邊框 + 底色 + badge |
| 台幣顯示 | 列表中省略前綴，僅顯示數字；統計欄才顯示 $ |
| 圖示 | Tabler outline icons（無 emoji 作為 UI 圖示） |

## 競品視覺現況

| 競品 | 主色調 | 視覺感受 | 敘事核心 |
|------|--------|----------|----------|
| **Tricount** | 深色背景、綠色強調 | 金融科技感、功能導向、偏冷 | 「把帳算清楚」 |
| **Splitwise** | 白底、綠橘配色 | 乾淨、簡潔、工具性強 | 「欠錢還錢」 |

**市場空缺**：兩大競品都是「財務工具」美學，沒有任何一款有溫度、有記憶感、有旅遊情境的視覺語言。這是我們的機會。

## 三個視覺方向選項

---

### 方向 A：旅遊日記（推薦）
**核心敘事**：「記下旅行的每一刻，帳只是其中一部分」

**視覺特徵**
- 色調：暖白底、奶油米色、橘棕點綴、柔和陰影
- 字體：主標題帶手寫感，內文清晰無襯線
- 每筆消費像日記條目——有日期章、有 emoji 圖示、可附照片
- 照片在介面中佔有一席之地，不是隱藏功能

**情感傳達**：溫暖、私密、有回憶感  
**類比參考**：實體旅遊日記、Instagram Stories 的敘事感  
**最適合**：Rozi 的使用情境（家庭旅遊、想回憶買了什麼）

---

### 方向 B：清爽旅遊工具
**核心敘事**：「旅遊記帳，清爽搞定」

**視覺特徵**
- 色調：白底、天藍、薄荷綠、清晰層次
- 字體：現代無襯線，工整清晰
- 功能優先，資訊密度適中
- 照片為輔，帳務為主

**情感傳達**：輕盈、高效、現代感  
**類比參考**：Notion、Monzo  
**最適合**：功能完整、擴大用戶群後的版本

---

### 方向 C：家人共享記憶
**核心敘事**：「一家人的旅行，每分每毫都值得記下」

**視覺特徵**
- 色調：珊瑚橘、暖黃、深藍點綴
- 強調成員存在感（大頭貼、emoji 識別）
- 家庭感設計語言，溫暖但不幼稚
- 結算畫面強調「大家一起」的感覺

**情感傳達**：溫馨、凝聚感、家庭專屬  
**類比參考**：BeReal、Family Album  
**最適合**：Phase 2 擴大到更多家庭用戶

---

## UX Lead 建議

**Phase 1 選方向 A（旅遊日記）**，原因：
1. 直接對應 Rozi 的核心需求（記憶 + 帳務）
2. 市場上完全空白，差異化最強
3. 與 Phase 2 的商業化路線（吸引更多用戶）相容
4. 「日記」敘事讓用戶有回來看的理由，不只是對帳

方向 B 可做為 UI 細節的參考（清爽、功能清晰），融入方向 A 的框架。  
方向 C 留待 Phase 2 做家庭版品牌延伸。
